# npm install -S -D -g 有什么区别

	npm install name -S   即   npm install name --save    写入dependencies
	npm install name -D   即   npm install name --save-dev 写入devDependencies
	npm install module_name -g 全局安装(命令行使用)

# dependencies与devDependencies有什么区别呢？

在package.json里 有dependencies和devDependencies两个
dependencies 是需要发布到生产环境的
devDependencies 里面的插件只用于开发环境，不用于生产环境
